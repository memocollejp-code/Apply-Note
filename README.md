# アプリ開発ログ

スマホ（Android / iOS）で使えるアプリ開発管理PWAです。

## 機能

- **修正管理** — 修正したい → 修正済み → 公開待ち → 再現確認 → 完了 のステータス管理
- **開発ノート** — アプリごとのメモ
- **プロンプト保管** — よく使うAIプロンプトをコピーしやすい形で保存
- **スクリーンショット** — アプリのスクショを記録
- **公開履歴** — バージョンと変更内容を記録
- **アイデア保留庫** — 今はやらないアイデアをストック
- **バックアップ / 復元** — JSONエクスポート・インポート
- **オフライン対応** — Service Worker によるキャッシュ

## 使い方

GitHub Pages で公開して、スマホのブラウザでアクセスしてください。

### GitHub Pages の設定

1. このリポジトリを Fork またはそのまま使う
2. Settings → Pages → Branch: `main` / `/(root)` で Save
3. 表示されたURLをスマホで開く
4. ブラウザのメニューから「ホーム画面に追加」でPWAとしてインストール

## ファイル構成

```
index.html      メインアプリ
sw.js           Service Worker（オフライン対応）
manifest.json   PWAマニフェスト
icon-192.png    アイコン192px（任意）
icon-512.png    アイコン512px（任意）
```

## データ保存

IndexedDB を使用してブラウザ内に保存されます。アンインストールやブラウザのデータ削除で消えるため、定期的にJSONバックアップを推奨します。
